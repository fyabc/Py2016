# Py2016

The project of Python 2016 Course.

This is a mini game **Shift**, written by **Python**, using *pygame* library.

## Requirements

Python 3 (3.4 / 3.5)
Pygame 1.9.2a0

## Start

Open the *Py2016* directory, and run *main.py*.
You can also run *main.py* in other directories.

Such as this:

```     
    cd Py2016
    python main.py
```
